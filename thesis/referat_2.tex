\chapter*{Общая характеристика работы}

Магистерская диссертация, \pageref{lastpage}~с., 28 рис., 7 табл., 25 источников

\bigskip

\noindent ОПТИМАЛЬНОЕ УПРАВЛЕНИЕ, УПРАВЛЕНИЕ С ПРОГНОЗИРУЮЩЕЙ МОДЕЛЬЮ,  НЕЙРОННАЯ СЕТЬ, НЕЙРОСЕТЕВОЙ РЕГУЛЯТОР, ОБУЧЕНИЕ С ПОДКРЕПЛЕНИЕМ

\bigskip

Объектом исследования магистерской диссертации являются задачи стабилизации нелинейных динамических систем, на управляющие воздействия и траектории которых наложены жесткие ограничения, и связанные с рассматриваемой задачей схемы управления по прогнозирующей модели, реализованные на основе нейронных сетей и  методов обучения с подкреплением.

Целью работы является построение нейросетевых регуляторов, гарантирующих асимптотическую устойчивость нелинейной системы, регуляторов на основе методов обучения с подкреплением.

Основной результат представлен алгоритмами управления на основе  нейронных сетей, позволяющими (по результатам экспериментов) повысить производительность схемы управления с прогнозирующей моделью  в $20$ раз.

Работа состоит из трех глав. В первой главе определяются основные понятия и результаты в области управления с прогнозирующей моделью. Во второй главе рассматриваются методы оффлайн дизайна обратных связей и их аппроксимаций в системах с прогнозирующей моделью. В третьей главе представлены результаты построения нейросетевого регулятора для нелинейных систем с ограничениями, результаты построения регулятора с помощью методов обучения с подкреплением для систем с неявной динамикой, а также проведено сравнение предлагаемых подходов с классическими алгоритмами, основанными на решении задач оптимального управления онлайн.

Новизна представленной работы состоит в построении нейросетевых регуляторов совместно с построением областей притяжения и областей насыщенных управлений, использовании результатов теории робастного управления для гарантированной стабилизации нелинейной системы, исследовании влияния различных техник сэмплирования точек для построения обучающих выборок, что помогает сократить время обучения без потерь точности.

Областями применения разработанных алгоритмов являются  прикладные задачи, решаемые в рамках теории управления с прогнозирующей моделью и возникающие в робототехнике, химической промышленности, транспортных системах, системах управления беспилотными аппаратами и др.


\chapter*{АГУЛЬНАЯ ХАРАКТАРЫСТЫКА РАБОТЫ}

Магістарская дысертацыя, \pageref{lastpage}~с., 28 мал. , 7 табл., 25 крыніц

\bigskip

\noindent АПТЫМАЛЬНАЕ КIРАВАННЕ, КIРАВАННЕ ПА ПРАГНАЗУЮЧАЙ МАДЭЛI, НЕЙРАСЕТКАВЫ РЭГУЛЯТАР, НАВУЧАННЕ З ПАДМАЦАВАННЕМ

\bigskip

Аб'ектам даследавання з'яўляюцца алгарытм кіравання па прагназуючай мадэлi, нейрасетевы рэгулятар і метады навучання з падмацаваннем.

Мэтай дадзенай працы з'яўляецца ужыванне нейрасетевых рэгулятараў, рэгулятараў на аснове метадаў навучання з падмацаваннем для павышэння прадукцыйнасцi сістэм па прагназуючай мадэлi.

У выніку даследаванняў было атрымана, што прадукцыйнасць сістэмы з нейрасеткавым рэгулятарам зменшыла час выканання аднаго кроку MPC у $20$ раз. Тэхніка стахастычнага сэмплавання з павелічэннем шчыльнасці вакол пачатку каардынат зменшыла лік пунктаў, патрэбных для пабудовы трэнеравальнага набору на $17 \%$, а значыць і скараціла час на навучанні. Сярэдні час выканання аднаго кроку MPC для стандартнай працэдуры складаў $1240$ мс, а пры выкарыстанні нейрасеткавага рэгулятара составлят $52$ мс.

У структуры працы вылучаюцца тры часткі. У першай частцы вызначаюцца асноўныя паняткі і вынікі ў вобласці кіравання з прагназавальнай мадэллю. У другой частцы разглядаюцца метады афлайн дызайну ў сістэмах з прагназавальнай мадэллю. У трэцяй частцы пададзены вынікі пабудовы нейрасеткавага рэгулятара для нелінейных сістэм, вынікі пабудовы рэгулятара з дапамогай метадаў навучання з падмацаваннем для сістэм з невыяўнай дынамікай, а таксама прыведзены вынікі параўнання для гэтых метадаў.

Навізной дадзенай працы з'яўляецца даследаванне нейрасеткавых рэгулятараў супольна з пабудовай абласцей прыцягнення і абласцей насычаных кіраванняў. Быў даследаваны ўплыў розных тэхнік сэмплавання кропак для пабудовы трэнеравальных датасэтаў, якія дапамаглі скараціць час навучання без страт дакладнасці.

\chapter*{GENERAL CHARACTERISTICS OF THE WORK}

Master thesis, \pageref{lastpage} ~ p., 28 figures, 7 tables, 25 sources

\bigskip

\noindent OPTIMAL CONTROL, MODEL PREDICTIVE CONTROL, NEURAL NETWORK REGULATOR, REINFORCEMENT LEARNING

\bigskip

The object of the research is the model predictive control algorithm, a neural network controller and reinforcement learning methods.

The purpose of this work is the use of neural network regulators, regulators based on reinforcement learning methods to improve the performance of model predictive control systems.

As a result of research, it was obtained that the system performance with a neural network regulator reduced the execution time of one MPC step by $ 20 $ times. The stochastic sampling technique with increasing density around the origin reduced the number of points needed to build a training set by $ 17 \% $, and therefore reduced the time for training. The average execution time for one MPC step for a standard procedure was $ 1240 $ ms, and when using a neural network controller it is $ 52 $ ms.

There are three chapters in the work structure. The first chapter defines the basic concepts and results in the model predictive control field. The second chapter discusses offline design methods in model predictive control systems. The third chapter presents the results of building a neural network controller for nonlinear systems, the results of building a controller using reinforcement learning methods for systems with implicit dynamics, and also shows the comparison results for these methods.

The novelty of this work is the study of neural network regulators together with the construction of domains of attraction and areas of saturated controls. The effect of different sampling techniques for building training datasets, which helped reduce training time without loss of accuracy, was investigated.
